# Ripple Status Bar

An [Atom](https://atom.io/) package to keep an eye on the latest [XRP](https://ripple.com/xrp/) price while you code!

The price is provided by [CoinMarketCap](https://coinmarketcap.com/)'s API, expressed in USD.
